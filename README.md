# TermLearn

字符终端风格的 AI 学习桌面应用。课程不写死，用户可以输入课程名称或导入考纲，由 AI 生成课程结构、章节内容、练习题、评分和讲解。

## 功能特性

- 任意课程：支持课程名称或 `.txt/.md` 考纲文本导入。
- 假终端交互：通过命令完成导入、学习、出题、答题、追问和布局切换。
- 动态课程生成：课程画像、知识点、题型和评分标准都由 AI 基于输入内容生成。
- AI 厂商切换：支持 GLM 和 DeepSeek，默认使用 GLM。
- 模型选择：AI 面板可获取当前厂商模型列表并选择模型。
- 多模态答案：支持粘贴图片作为答案附件，仅支持多模态模型；文本模型会在提交前拦截。
- 流式输出：可在 AI 面板开启或关闭，开启后 AI 请求使用流式响应。
- 中英文界面：底部状态栏支持全局语言切换。
- 本地单用户：课程、练习和布局数据保存在本机；API Key 使用 Electron 安全存储加密保存。

## 快速开始

安装依赖：

```bash
npm install
```

启动开发版桌面应用：

```bash
npm run dev
```

构建：

```bash
npm run build
```

类型检查：

```bash
npm run typecheck
```

## AI 配置

打开右侧 AI 面板：

1. 选择 AI 提供者，默认是 `GLM`。
2. 保存对应厂商的 API Key。
3. 点击刷新模型列表。
4. 选择要使用的模型。
5. 如需粘贴图片答案，请选择支持多模态的 GLM 模型。

API Key 按厂商分别保存，不会写入课程数据、日志或源码。

## 常用命令

```bash
help
```

查看全部命令和说明。

```bash
course create 考研英语
```

直接用课程名称创建课程，不需要考纲。

```bash
syllabus import
```

打开导入窗口，可输入课程名称，也可粘贴完整考纲。

```bash
course list
course enter <courseId>
```

查看课程并进入课程。进入课程后，后续命令可省略课程 id。

```bash
lesson start
lesson start 长难句分析 入门讲解 多给例句
lesson start 生成一节矩阵初等变换课程
```

生成当前课程的一节学习内容。`lesson start` 后可以追加自然语言要求，让 AI 按指定主题、范围、深度、讲解方式、示例类型或练习形式生成课程。

```bash
quiz start
quiz start 阅读理解 中等难度
quiz start 生成一道选择题
```

生成练习题。`quiz start` 后可以追加自然语言要求，让 AI 按指定题型、范围、难度或形式出题。

```bash
answer <你的答案>
```

提交当前题目答案并请求 AI 评分。存在待答题时，也可以直接输入答案。

```bash
explain
```

讲解最近一次答题记录，并进入追问模式。追问模式下可直接输入问题。

```bash
explain exit
exit
quit
```

退出追问模式。

```bash
layout focus
layout restore
```

切换纯终端模式和完整工作台。

```bash
clear
```

清空当前终端显示区。

## 交互细节

- `Tab`：自动补全命令。
- 上下方向键：切换历史命令。
- 图片粘贴：粘贴图片后会作为下一次答案附件提交；点击附件可移除。
- `clear`：只清空当前显示区，不重新输出启动 banner。
- 终端启动时只显示字符版 `TermLearn` banner。

## 数据与安全

- 应用数据保存在 Electron 的 `userData` 目录。
- API Key 使用 Electron `safeStorage` 加密后按厂商单独保存。
- 渲染进程不直接接触明文 Key，AI 请求由主进程代理。
- 图片附件只随本次答题请求发送给当前选择的 AI 模型。

## 当前边界

- 第一版是本地单用户桌面应用，不包含账号、云同步或多人协作。
- 模拟考试命令目前会提示使用 `quiz start` 逐题练习，完整考试流程后续可扩展。
- 课程质量取决于输入的课程名称、考纲质量和所选 AI 模型能力。
- 不为英语、编程、数学等课程写特殊逻辑，所有课程走同一套动态生成流程。
